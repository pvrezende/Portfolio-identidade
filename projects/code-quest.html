CODE QUEST

O Code Quest é uma experiência interativa em estilo RPG criada para unir programação, gamificação e evolução pessoal. O usuário cria seu personagem, escolhe uma classe de programador e inicia uma jornada baseada em desafios e progresso.

OBJETIVO

  • Transformar conceitos de programação em uma experiência divertida.
  • Permitir a criação de personagem e escolha de classe.
  • Aplicar interações e regras utilizando JavaScript.
  • Criar uma identidade visual inspirada em jogos de RPG.

VISÃO GERAL

O projeto utiliza elementos de jogos para tornar a navegação mais envolvente. A interface conduz o usuário por uma experiência temática, combinando formulários, escolhas, feedback visual e comportamento interativo.

TELA DO PROJETO

UMA JORNADA GAMER PARA DESENVOLVEDORES

A tela apresenta a criação do personagem, permitindo informar o nome do herói, selecionar uma classe de programador e definir o poder de estudo.

A proposta visual utiliza cores neon, tipografia temática e elementos de interface inspirados em RPG para criar uma experiência diferente de um formulário convencional.
