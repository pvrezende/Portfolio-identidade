CASE STUDY • FINANÇAS PESSOAIS

CONTROLE DE GASTOS

O Controle de Gastos é uma aplicação criada para ajudar na organização financeira pessoal. O sistema reúne despesas, receitas, metas e planejamento em uma interface centralizada, permitindo que o usuário acompanhe melhor sua vida financeira e tome decisões com mais clareza.

OBJETIVO

  • Centralizar receitas, despesas e informações financeiras.
  • Ajudar o usuário a acompanhar metas e planejamento pessoal.
  • Facilitar a visualização da situação financeira atual.
  • Oferecer uma experiência simples, organizada e responsiva.

VISÃO GERAL

A aplicação combina uma interface moderna com back-end, autenticação e persistência de dados. O projeto foi pensado para transformar informações financeiras em dados fáceis de consultar e acompanhar no dia a dia.

TELA DO PROJETO

GESTÃO FINANCEIRA CENTRALIZADA

A interface foi desenvolvida para permitir acesso rápido às informações financeiras e às principais ações do sistema. O visual escuro reforça a identidade do projeto e mantém os elementos importantes em destaque.

O usuário pode acessar a aplicação publicada para conhecer a experiência, navegar pelas funcionalidades disponíveis e visualizar o projeto funcionando.
