CASE STUDY • APRESENTAÇÃO PROFISSIONAL

CURRÍCULO WEB

O Currículo Web é uma aplicação criada para apresentar minha trajetória profissional de forma digital, organizada e responsiva. O projeto reúne informações pessoais, tecnologias, experiências, formação e links profissionais em uma interface moderna.

OBJETIVO

  • Apresentar informações profissionais de forma clara.
  • Permitir alternância de perfil e idioma.
  • Oferecer tema claro e escuro.
  • Disponibilizar uma versão pronta para impressão em PDF.

VISÃO GERAL

O projeto transforma o currículo tradicional em uma experiência web. A interface permite consultar diferentes perfis profissionais e facilita o acesso a contatos, habilidades, formação e histórico de atuação.

TELA DO PROJETO

CURRÍCULO DIGITAL, RESPONSIVO E INTERATIVO

A página apresenta dados profissionais, contato, tecnologias, formação e experiências em blocos organizados. O usuário também pode alternar tema, idioma e perfil profissional.

A funcionalidade de impressão permite gerar uma versão em PDF, unindo apresentação online e documento profissional em um único projeto.
